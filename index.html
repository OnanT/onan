Kalinago Territory Initiative

A descendant's digital contribution to preserving and honoring Kalinago heritage and culture.

The Kalinago, once known as Caribs, are the Indigenous people of the Lesser Antilles. The Kalinago Territory in Dominica remains one of the few legally recognized Indigenous communities in the Caribbean.

Cultural Preservation Goals:

  • Document Kalinago language, traditions, and history
  • Create digital archive of cultural artifacts
  • Develop educational resources for schools
  • Support community economic development
  • Promote Kalinago arts and crafts globally

GreyWolf Project

Automated Reconnaissance Framework

GreyWolf is a modular bug bounty recon framework tailored for automated API and web asset discovery. It integrates tools like Kiterunner, nuclei, ffuf, and others, and is built for extensibility.

Key Features:

  • Subdomain Enumeration (AssetFinder, Amass)
  • Advanced Fuzzing with custom wordlists
  • API Discovery and Testing
  • Vulnerability Scanning (nuclei)
  • Modular Architecture for easy extension
